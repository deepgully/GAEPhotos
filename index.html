GAEPhotos by deepgully

Web photo albums based on GAE, demo on http://gaephotos.deepgully.com

Dependencies

GAE HRD, Python2.7

Install

1. Create GAE Python HRD application, HRD is required for version 1.01. 2. Download latest package from http://code.google.com/p/gaephotos/downloads/list 3. Unzip to local folder. 4. Change the application id in app.yaml to your app id. application: gaephotos2 5. Run update.bat. (make sure appcfg.py in your system path) 6. Done! Wait until DB indexes created(several minutes or days). 7. Visit xxxxxx.appspot.com and upload your photos, Enjoy! PS. 中国大陆用户请绑定域名使用.
